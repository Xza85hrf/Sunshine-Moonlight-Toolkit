# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-XX-XX

### Added
- Initial public release
- **Setup-Toolkit.ps1** - Interactive first-run configuration wizard
- **Toolkit-Launcher.bat** - Menu-driven launcher for all tools
- **Apply-OptimalSettings.ps1** - Moonlight profile configurator with 5 presets
- **Test-Connection.ps1** - Connection quality tester
- **Optimize-SunshineHost.ps1** - Host firewall and network configuration
- **Fix-Tailscale.ps1** - Tailscale troubleshooter for stuck connections
- **Stream-Monitor.ps1** - Real-time stream quality monitoring
- **Sunshine-Moonlight-Optimizer.ps1** - Comprehensive diagnostics
- Central configuration system (config.json)
- GitHub issue and PR templates
- MIT License

### Streaming Profiles
- **Performance**: 720p/60fps @ 20 Mbps
- **Balanced**: 1080p/60fps @ 50 Mbps
- **Quality**: 1080p/60fps @ 80 Mbps
- **Ultra**: 1080p/120fps @ 100 Mbps (LAN recommended)
- **Competitive**: 1080p/120fps @ 60 Mbps (V-Sync off)

### Supported Ports
- TCP: 47984, 47989, 47990
- UDP: 47998, 47999, 48000, 48010
